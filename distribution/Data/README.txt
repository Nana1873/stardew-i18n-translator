Stardew i18n Translator portable data

The application stores its local data in this folder:

- settings.json: selected game/mod folders, language, and local-AI settings
- glossary.json: optional locally generated Stardew glossary
- translations/: saved translation work and automatic state backups

Keep this Data folder beside stardew-i18n-translator.exe.
Copy the complete application folder to move your work to another computer.

The saved Stardew Valley and Mods paths are absolute and may need to be selected
again when the folder layout differs on the other computer.
