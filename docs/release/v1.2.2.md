# Stardew i18n Translator v1.2.2

This maintenance release tightens the editing workflow, removes an unreliable
Nexus search shortcut, and reduces manual project and release administration.

## User-Facing Changes

- Added **Open Mods Folder** to the mod-list context menu.
- Removed inline translation editing from the string table. Editing now always
  uses the validated String Editor dialog.
- The toolbar string search is shown only while a mod is open, avoiding the
  broken dashboard-to-workspace focus handoff.
- Removed the browser-based **Search Translation on Nexus** action because its
  search results were unreliable. Broader Nexus integration is deferred
  indefinitely.

## Project Maintenance

- Active work and release commitments now live in GitHub Issues and Milestones
  instead of duplicated roadmap documents.
- Added synchronized version update and validation commands.
- CI now checks local Markdown links, version references, formatting, and pull
  request documentation policy.
- Tagged releases generate categorized pull-request notes automatically while
  retaining these curated highlights.

## Upgrade Notes

- Replace the previous portable application files with the v1.2.2 package.
- Keep the existing `Data/` directory to preserve settings, glossary data, and
  translation progress.
- No data migration is required.

The application remains a portable, local-first Windows tool. It does not add
cloud credentials, telemetry, or Nexus API access.
