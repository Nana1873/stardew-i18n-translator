# Stardew i18n Translator v1.2.0

This release improves visibility into stale translation files and gives users
direct control over local diagnostic logging.

## Highlights

- Scans now show an **Optional cleanup** section for unused translation keys
  that still exist in a target-language JSON file but no longer exist in
  `default.json`.
- Each entry identifies the mod, target file, and unused key.
- The app explains that SMAPI ignores these keys. They are safe to keep or
  remove, are not imported, do not affect progress, and do not block export.
- **Settings → About** now includes an **Enable local diagnostic logging**
  switch.
- Logging remains enabled by default and can be disabled immediately without
  deleting existing logs.

## Privacy

Logs are local, rotating, and size-capped. There is no telemetry, crash upload,
account, or remote log destination. The logging preference is stored only in
the portable `Data/settings.json`.

## Installation

Download and extract the portable Windows ZIP to a writable folder, then run
`stardew-i18n-translator.exe`. Keep the adjacent `Data` folder with the
executable when moving the application.

The executable is currently unsigned, so Windows SmartScreen may show an
unknown-publisher warning.
