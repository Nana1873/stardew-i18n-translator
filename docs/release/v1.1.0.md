# Stardew i18n Translator v1.1.0

V1.1 focuses on faster editing, safer exports, and verified multilingual
workflows while keeping the application fully portable.

## Highlights

- Edit short, single-line translations directly in the string table.
- Configure keyboard shortcuts, detect conflicts, and reset defaults.
- Drag external LLM batch-result JSON files directly onto the application.
- Block a complete mod export before any file or backup is written when
  protected-token counts differ.
- Use all 11 advertised target languages through verified Unicode-safe scan,
  edit, batch, and export workflows.
- Import an existing Portuguese `pt-BR.json` file and export canonically to
  `pt.json`.

## Data Safety

Saved working translations are now isolated by target language under
`Data/language-state/<code>/translations/`. Existing v1.0.x state in
`Data/translations/` is moved once into the language configured during the
upgrade. Switching languages no longer risks showing or exporting work from a
different language.

Copy the complete portable application folder before upgrading when you want
an additional backup of your current workspace.

## Other Improvements

- Added an About page with version, license, author, repository, and technology
  information.
- Updated the GitHub README with the current portable workflow and real app
  screenshots.
- Added GPL-3.0-or-later licensing metadata.
- Improved CI and release build times.

## Installation

Download and extract the portable Windows ZIP to a writable folder, then run
`stardew-i18n-translator.exe`. Keep the adjacent `Data` folder with the
executable when moving the application.

The executable is currently unsigned, so Windows SmartScreen may show an
unknown-publisher warning.
