# Stardew i18n Translator v1.1.1

A small diagnostics hotfix: the app now writes a local log file so problems can
be reported with something a maintainer can actually act on.

## Highlights

- The app writes a **local diagnostic log** to `Data/logs/` beside the
  executable. It captures backend and frontend errors as you use the app.
- The log **rotates** and is **size-capped**, so it never grows unbounded inside
  the portable folder.
- New **Settings → About → Open logs folder** button opens the folder so you can
  attach the newest log to a GitHub bug report.

## Privacy

Logging is fully local — there is no network upload, no telemetry, and no
accounts, consistent with the rest of the app. The log is only shared if **you**
attach it to a report. It may contain local folder paths, so remove anything
private before sharing.

## Installation

Download and extract the portable Windows ZIP to a writable folder, then run
`stardew-i18n-translator.exe`. Keep the adjacent `Data` folder with the
executable when moving the application.

The executable is currently unsigned, so Windows SmartScreen may show an
unknown-publisher warning.
