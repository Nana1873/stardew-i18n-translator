# Stardew i18n Translator v1.3.0

This release adds a compact workflow for preparing mod translations for
sharing while keeping the app local and portable.

## Highlights

- Export and import outcomes now stay available in a persistent result tray
  without blocking the translation workspace.
- **Build Release ZIP** creates an installable translation archive while
  preserving the selected mod package's folder structure.
- **Translation Notes** generates short, localized text with the package
  version, language, coverage, installation instructions, and compatibility
  fallback.
- External-LLM batch exports include the complete four-step handoff and an
  exact prompt that can be copied directly.
- If Microsoft WebView2 is missing, the native launcher explains the
  requirement and links to the official Microsoft download page.

## Upgrade Notes

- Replace the previous portable application files with the v1.3.0 package.
- Keep the existing `Data/` directory to preserve settings, glossary data, and
  translation progress.
- Microsoft Edge WebView2 Runtime must be installed. It is already present on
  most supported Windows systems.
- No data migration is required.

The application remains a portable, local-first Windows tool without
telemetry, cloud credentials, Nexus API access, or automatic uploads.

## Verification Note

The complete frontend, Rust, and documentation checks passed in GitHub Actions
on the exact release commit. Release-script, packaging, and extracted-archive
