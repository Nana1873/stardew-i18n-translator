# Agent Workflows

This directory contains specialized workflow manuals and templates for the AI coding agents that build and maintain this repository:

* **[Google Antigravity Workflow](file:///docs/agents/antigravity-workflow.md):** Best for planning-heavy tasks, system integration, validation, and architecture-sensitive issues.
* **[OpenAI Codex Workflow](file:///docs/agents/codex-workflow.md):** Best for isolated components, refactoring, writing unit tests, and PR-style implementations.
* **[Claude Code Workflow](file:///docs/agents/claude-code-workflow.md):** Best for local workspace analysis, multi-file edits, milestone tasks, and local command execution.

## Handing Off Work

When you finish a task, always write a handoff summary using the [Handoff Template](file:///docs/agents/handoff-template.md) at the end of your run to ensure a seamless transition for the next agent or developer.
