Type
Segoe UIUI · system on Windows, no webfont needed
JetBrains Monokeys, tokens, shortcuts
Min body 11px · table rows 30px · numerals use tabular-nums so progress/counts don't jitter.
Why these tokens
  • Three warm surface steps + one well (surface-3) give depth without shadows the GPU has to blur on a 17k-row table.
  • Gold is the only brand hue — never used for a status — so selection and "needs review" can't be confused.
  • Status carries 3 signals (hue + glyph + edge); any one alone is sufficient → colorblind-safe.
02 · ROW STATES Solid edges, never full-row tint blur

Rationale: status lives in a fixed-width chip + a 3px left edge. Hover and selection are the only things that tint the row background — so statuses never smear together down 300 rows.
